# Wanderlust 🌍

A full‑stack travel listing web application where users can explore destinations and create their own listings. The project is inspired by modern travel and property listing platforms and is built to practice real‑world backend and full‑stack development.

---

## 🚀 Project Overview

Wanderlust is a dynamic web platform that allows users to browse travel listings, add their own destinations, and manage listings securely. The application focuses on building a scalable backend, implementing authentication, and performing full CRUD operations.

This project is currently under development and new features are being added continuously.

---

## ✨ Features

### User Authentication

* Secure signup and login functionality
* Session‑based authentication
* Authorization middleware to protect routes

### Listings Management

* Create new travel listings
* View all available listings
* Edit existing listings
* Delete listings

### Dynamic Rendering

* Server‑side rendering using EJS templates
* Dynamic pages displaying data from the database

### Flash Messages

* Success and error messages for better user experience

### MVC Architecture

* Structured project using Model‑View‑Controller pattern
* Clean and maintainable backend structure

## ⚙️ Installation

Follow these steps to run the project locally:

1. Clone the repository

```
git clone https://github.com/vishal0838/Wanderlust.git
```

2. Navigate into the project folder

```
cd Wanderlust
```

3. Install dependencies

```
npm install
```

4. Start the server

```
npm start
```

5. Open your browser and visit

```
http://localhost:8080/listings
```
